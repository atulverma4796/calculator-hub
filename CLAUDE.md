# Calculator Hub — Project Context

## What is this
A free online calculator hub with 15+ financial, math, health, and utility calculators. Modern, beautiful, mobile-first UI. No signup, no limits, ad-supported.

## Why this project
- Calculator.net gets 54M visits/mo with a design from 2005
- Finance AdSense CPC is $5-50/click — highest paying niche
- "EMI calculator" alone = 1M+ searches/mo in India
- Pure math + clean UI — no file processing, no APIs, no server cost
- AI-proof: people trust interactive calculators with sliders over asking ChatGPT

## Tech stack
- Next.js (App Router) + TypeScript + Tailwind CSS
- Vercel (free hosting)
- Client-side only — zero server cost, zero API calls
- Google AdSense (monetization) — placeholder until approved

## Design principles
- Premium, modern UI — glassmorphism, smooth animations, beautiful gradients
- Mobile-first — 60%+ traffic is mobile
- Each calculator = own page = own SEO target
- Interactive sliders + live results (not just input then submit then result)
- Charts/graphs for visual output (compound interest growth, loan amortization)

## Revenue model
- Google AdSense only — no paywalls, no premium tier
- Finance niche CPC: $5-50/click
